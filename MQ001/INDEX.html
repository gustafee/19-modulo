Existem 3 tipos de Midias queries o Midia Screen, Midia print e Midia Type

O Midia screen é a media query padrão, aplicada quando a página é exibida em tela (por exemplo, em um monitor de computador, tablet ou celular). Você usa esta media query para definir estilos que serão aplicados na visualização padrão da página web.

O midia Print é aplicada quando a página é impressa ou visualizada em modo de impressão no navegador. Você utiliza esta media query para ajustar os estilos para melhorar a legibilidade em papel, remover elementos não essenciais, etc.

Midia type é uma media query genérica que permite especificar uma ampla gama de características para aplicar estilos baseados no dispositivo ou na saída. Por exemplo, você pode usar características como largura de tela (width), altura de tela (height), orientação (orientation), resolução (resolution), entre outras.
